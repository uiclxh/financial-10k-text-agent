# Working Paper

**Auditing Predictive Information in SEC 10-K Text: Evidence from a 50-Firm U.S. Large-Cap Pilot**

[Read the working paper (PDF)](auditing_predictive_information_in_sec_10k_text.pdf)

The paper documents the v4 research design, rolling out-of-sample evidence, feature ablation, bootstrap inference, parser-quality review, portfolio diagnostics, and empirical claim boundaries.

The evidence supports exploratory volatility-ranking information. It does not establish formal tradable alpha or provide investment advice.
