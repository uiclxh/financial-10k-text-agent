# v0.9.0 Portfolio Variants

Release date: 2026-05-23

Portfolio construction upgrade for research-grade factor diagnostics. This
release extends the portfolio time-series layer from a single equal-weight
long-short variant to multiple construction variants when sector and market-cap
metadata are available.

## Changes

- Added optional `sector`, `industry`, and `market_cap` fields to prediction
  records.
- Propagated metadata features from the feature layer through model predictions.
- Added `portfolio_variant`, `weighting`, and `sector_neutral` fields to
  portfolio artifacts.
- Added automatic `equal_weight` portfolio construction.
- Added automatic `value_weight` portfolio construction when market cap is
  available.
- Added automatic `sector_neutral_equal_weight` construction when sector
  metadata is available.
- Added automatic `sector_neutral_value_weight` construction when both sector
  and market cap are available.
- Added sector-level ranking with balanced long and short exposure across
  eligible sectors.
- Added value-weight leg weights based on market cap within each selected leg.
- Added tests for portfolio variant discovery, value-weight sizing, and
  sector-neutral net exposure.

## Boundaries

This release implements portfolio-construction variants on top of event-window
label returns. It does not yet implement daily price-driven holding simulation,
borrow costs, capacity constraints, Alphalens-style tear sheets, or
multiple-testing adjustment reports.
