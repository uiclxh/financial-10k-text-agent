# v0.16.0 - 10-Company FMP/Alpha Applied Pilot

Release aligning the repository, README, and public artifacts around the current
10-company applied-grade SEC 10-K pilot.

## Changes

- Replaced the WRDS/Nasdaq-dependent public workflow with an
  FMP/Alpha/Yahoo-fallback data stack for the committed applied pilot.
- Replaced SONY with JPM so the main public universe remains a pure U.S. `10-K`
  panel.
- Added the 10-company public result package under
  `docs/results/10_company_public_fmp_alpha_2016_2025_v1/`.
- Published compact public artifacts for 100 SEC filings, 300 labels, and 896
  out-of-sample predictions.
- Reported 0 audit failures and 2 audit warnings.
- Reported 100% eligible OOS coverage.
- Added coverage waterfall outputs to separate raw label coverage from eligible
  OOS prediction coverage.
- Added mixed-source market-data boundary documentation for the FMP primary
  source and Yahoo fallback.
- Kept portfolio outputs diagnostic only; the current public run does not
  establish formal trading alpha.
- Updated working-paper positioning so the 10-company FMP/Alpha pilot is the
  current primary public package and the earlier 90-company run is a historical
  diagnostic package.

## Headline

The repository now presents the current public result as an applied-grade,
leakage-aware 10-company SEC 10-K experiment: audit-clean enough for a research
engineering demo, but explicitly not a CRSP/WRDS formal replication and not a
production trading-alpha claim.
