# v0.7.0 Research-Grade Event Calendar

Release date: 2026-05-22

Event-time alignment release for SEC filing studies. This version upgrades
event-date resolution from a fixed close-time heuristic to an exchange calendar
based resolver that handles holidays, weekends, pre-open filings, intraday
filings, after-close filings, and early closes.

## Changes

- Added `text_factor_lab.calendar` package.
- Added NYSE market-session lookup using `pandas-market-calendars`.
- Added `resolve_event_date(...)` for UTC event times.
- Added explicit event-date policies:
  - `pre_open`
  - `intraday`
  - `after_close`
  - `non_trading_day`
- Added early-close detection using the exchange schedule.
- Updated SEC EDGAR manifest construction to use the market-calendar resolver.
- Added manifest audit fields:
  - `raw_filing_date`
  - `acceptance_time_utc`
  - `market_open_utc`
  - `market_close_utc`
  - `is_trading_day`
  - `is_early_close`
  - `event_date_policy`
  - `resolved_event_date`
  - `resolved_event_time_version`
- Added tests for weekend, holiday, pre-open, intraday, after-close, and
  early-close event-date cases.

## Boundaries

This release improves event-time alignment for US equity filing studies. It does
not yet implement a full portfolio time-series backtest, multiple-testing
adjustments, or research-grade survivorship-free universe construction.
