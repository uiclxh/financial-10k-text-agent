# v0.11.0 Empirical Reports

Release date: 2026-05-23

Report-agent upgrade for clearer empirical-finance communication. This release
keeps the existing `report.md` and `report_summary.json` while adding dedicated
research-facing report artifacts.

## Changes

- Added `empirical_report.md`.
- Added `factor_card.md`.
- Added `appendix_tables.md`.
- Added structured interpretation policy to `report_summary.json`.
- Added economic interpretation and evidence-level text.
- Added empirical report sections for:
  - research design
  - data and universe construction
  - event-time alignment
  - text feature construction
  - label construction
  - prediction models
  - out-of-sample forecasting results
  - portfolio construction
  - factor backtest results
  - sector-neutral and value-weighted robustness
  - multiple-testing adjustment
  - audit results
  - economic interpretation
  - limitations
  - conclusion
- Added factor card summary for audit status, evidence level, best prediction,
  best backtest, and usage boundary.
- Added appendix tables for sample coverage, feature summary, OOS metrics,
  portfolio results, portfolio variants, multiple-testing results, and audit
  checks.
- Integrated the new artifacts into CLI report output.
- Integrated the new artifacts into orchestrator report-stage outputs.
- Added tests for generated empirical report, factor card, appendix tables, and
  e2e smoke output.

## Boundaries

This release improves research communication and artifact review. It does not
turn the demo universe into a survivorship-free research universe, and it does
not add Deflated Sharpe, CPCV/PBO, bootstrap intervals, clustered errors, or
daily price-driven holdings.
