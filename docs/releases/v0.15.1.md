# v0.15.1 - Public Applied-Grade 10-K Pilot

Compact public result package and README refresh for the fixed 10-company
FY2016-FY2025 SEC 10-K pilot.

## Changes

- Added a GitHub-friendly public result package under
  `docs/results/10_company_public_fmp_alpha_2016_2025_v1/`.
- Published representative artifacts: factor card, empirical report, audit
  report, coverage waterfall, evaluation metrics, model and feature manifests,
  vocabulary manifest, multiple-testing report, specification registry, labels,
  predictions, and selected portfolio diagnostics.
- Excluded raw SEC filings, full price panels, full feature matrices, and full
  daily portfolio return series from the public package.
- Rewrote the root README to make the repository purpose, pipeline, latest
  result, and limitations clear in under a minute.
- Updated `docs/results/README.md` to point to the latest 10-company public
  package while keeping earlier large-universe summaries as historical
  diagnostics.
- Added checksums for committed public artifacts.
- Kept the conclusion boundary explicit: exploratory forecasting evidence,
  diagnostic portfolio results, and no formal trading-alpha claim.

## Headline

The latest public run has 100 SEC 10-K filings, 300 labels, 896 OOS
predictions, 100% eligible OOS prediction coverage, 0 audit failures, 2 audit
warnings, and 472 tested specifications.

The public pilot validates the leakage-aware research workflow. It does not
establish a production trading strategy.
