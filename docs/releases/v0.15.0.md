# v0.15.0 - Preregistered Portfolio Inference And 90-Company Results

v0.15.0 upgrades the research inference layer and publishes compact summaries
from the public 90-company SEC 10-K large-universe run.

## Changes

- Adds validation-selected portfolio direction.
- Adds target-aware portfolio policy:
  - `none`
  - `long_low_vol`
  - `risk_scaled`
- Forces volatility-target portfolios to use low predicted volatility on the
  long side when target-aware policy is enabled.
- Adds risk-scaled long/short weights using inverse predicted volatility.
- Adds monthly common-rebalance portfolio aggregation at `ALL_SPLITS`.
- Changes primary portfolio p-value to use the `ALL_SPLITS` OOS return-series
  Newey-West t-stat.
- Upgrades `specification_registry.json` to `specification-registry-v1`.
- Marks primary prediction and portfolio rules as preregistered rules.
- Records `p_value_method` in tested specifications.
- Adds compact 90-company result summaries under `docs/results/`.

## Result Summary

- 90 companies with SEC 10-K documents.
- 883 documents, 2,649 labels, and 8,236 predictions.
- Best prediction target: `realized_volatility_1_20`.
- Best ALL_SPLITS volatility predictor: `xgboost`, Rank IC `0.3899`.
- Preregistered primary prediction passes:
  - `realized_volatility_1_20 / ridge`
  - Rank IC `0.3335`
  - p-value about `6.85e-11`
- Preregistered primary portfolio does not pass:
  - Sharpe `-0.3602`
  - p-value `0.4598`

## Interpretation

The current evidence supports 10-K text as a volatility forecasting signal, not
yet as a formal cost-adjusted, sector-neutral, multiple-testing-adjusted trading
factor.
