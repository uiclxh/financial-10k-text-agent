# v0.1.0 Phase 1 - Foundation / Project Initialization

Release date: 2026-05-20

First foundation release for the Financial 10-K Text Agent. This version
establishes the Python package, experiment config contract, schema layer, and
initial run-management artifacts.

## Changes

- Added Python package scaffold under `src/text_factor_lab`.
- Added CLI entry point through `python -m text_factor_lab`.
- Added `mvp_v0.yaml` experiment config scaffold.
- Added Pydantic config and artifact schemas.
- Added `formal_run` / `exploratory_run` base concepts.
- Added run directory creation.
- Added `config_snapshot.yaml`.
- Added `run_status.json`.
- Added `failure_log.jsonl`.
- Added initial orchestrator contract.

## Boundaries

This release does not include SEC data ingestion, parsing, labels, features,
models, backtesting, or audit logic.
