# v0.1.0 MVP Workflow Release

Release date: 2026-05-20

This release documents the current MVP state of `Financial 10-K Text Agent`.
The repository now contains executable code, CLI commands, schemas, and tests
for Steps 1-12 of the global workflow.

## Completed Scope

- Project scaffold, config schema, and run manager.
- Fixed universe validation and quality reporting.
- SEC EDGAR 10-K metadata helpers.
- SEC 10-K section parser for Item 1, Item 1A, Item 3, and Item 7.
- Price CSV loader and event-label builder.
- Rolling-year train / validation / test split with leakage checks.
- Dictionary tone and TF-IDF feature layer with feature manifests.
- Model layer:
  - `historical_mean`
  - `industry_mean`
  - `ridge`
  - optional `xgboost`
- Evaluation layer:
  - RMSE
  - MAE
  - R2
  - directional accuracy
  - Pearson IC
  - rank IC
  - split-level and all-split aggregate records
- Event-based long-short backtest:
  - top/bottom quintile with minimum one name per leg
  - equal-weight returns
  - transaction-cost adjustment
  - Newey-West t-stat
  - Sharpe ratio
- Audit gate:
  - artifact presence
  - schema validation
  - coverage threshold
  - prediction-label alignment
  - split leakage failures
  - feature lookahead
  - TF-IDF train-window fit-scope
  - model-selection metric audit
  - tested-specifications disclosure
  - formal metadata gate

## Known Boundaries

This is still an MVP research pipeline. Remaining research-grade work includes:

- Full one-command orchestration from data acquisition through report.
- Sector-neutral and value-weighted portfolio construction.
- True portfolio time-series turnover instead of prototype event-level turnover.
- Multiple-testing adjustment report.
- Final report / factor card generation.
- FinBERT, LLM embedding, LSTM, and Transformer extensions.
- Earnings-call transcript ingestion and license gates.
- Credit-risk target construction.
