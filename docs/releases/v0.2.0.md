# v0.2.0 Phase 2 - Data, Parsing, Labels, Splits, Features

Release date: 2026-05-20

Data-preparation release for 10-K text factor research. This version adds the
core artifact builders needed before model training: universe validation, SEC
metadata, 10-K parsing, price labels, rolling splits, and text features.

## Changes

- Added fixed universe validation with ticker, CIK, sector, industry, mapping
  source, and delisting fields.
- Added SEC EDGAR CIK normalization, submissions helpers, archive URL helpers,
  and 10-K metadata handling.
- Added SEC 10-K parser for Item 1, Item 1A, Item 3, and Item 7.
- Added table-of-contents false-positive filtering.
- Added section span and SHA256 hash artifacts.
- Added price CSV loader.
- Added `log_return` and `simple_return`.
- Added `CAR_1_n`.
- Added `realized_volatility_1_n`.
- Added `realized_volatility_annualized_1_n`.
- Added explicit label failure records.
- Added rolling train / validation / test splits.
- Added embargo leakage reports.
- Added dictionary tone features.
- Added section-level and full-text TF-IDF features fit on train windows only.
- Added `feature_manifest.json` and `vocabulary.json`.

## Boundaries

This release prepares model-ready artifacts but does not claim formal predictive
results. Large raw filings, licensed transcripts, CRSP/Compustat extracts, and
other restricted datasets should not be committed to GitHub.
