# v0.4.0 Step 13 - Report Agent

Release date: 2026-05-20

Report Agent release for completed MVP runs. This version turns audited model,
evaluation, and backtest artifacts into a reproducible Markdown report and a
machine-readable JSON summary.

## Changes

- Added `text_factor_lab.reports` package.
- Added report-generation CLI support.
- Added `report.md` output.
- Added `report_summary.json` output.
- Added executive summary, research question, data/sample, label, feature,
  model, prediction, backtest, robustness, audit, failure, conclusion, and
  reproducibility sections.
- Added automatic best prediction metric summary.
- Added automatic top backtest summary.
- Added audit failure handling:
  - `audit_status=fail` blocks standard report generation.
  - diagnostic-only reports are available when explicitly allowed.
- Added run status update to `reported` when a pass/warn audit report is
  generated.

## Boundaries

This release does not add an interactive dashboard, cloud deployment, or full
production report styling.
