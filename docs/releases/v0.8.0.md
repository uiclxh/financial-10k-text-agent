# v0.8.0 Portfolio Time-Series Backtest

Release date: 2026-05-22

Portfolio construction release for factor evaluation. This version keeps the
legacy event-level backtest summary while adding explicit portfolio weights,
portfolio return series, turnover, costs, exposures, and portfolio-level
performance metrics.

## Changes

- Added portfolio weight schema.
- Added portfolio return schema.
- Added portfolio metric schema.
- Added `portfolio_weights.jsonl`.
- Added `portfolio_returns.jsonl`.
- Added `portfolio_metrics.json`.
- Added equal-weight dollar-neutral top/bottom portfolio construction.
- Added rebalance-level:
  - factor rank
  - quantile
  - side
  - normalized weight
  - previous weight
  - trade weight
- Added realized turnover from actual weight changes.
- Added transaction-cost-adjusted portfolio returns.
- Added exposure diagnostics:
  - long exposure
  - short exposure
  - gross exposure
  - net exposure
  - active position count
- Added portfolio-level metrics:
  - cumulative return
  - annualized return
  - annualized volatility
  - Sharpe ratio
  - max drawdown
  - hit rate
  - average turnover
  - average gross exposure
  - average net exposure
- Extended `evaluate-models` and the local orchestrator to write portfolio
  artifacts.
- Added tests for rebalance turnover and end-to-end portfolio artifact output.

## Boundaries

This release introduces a portfolio time-series artifact layer based on label
window returns. It does not yet implement daily price-driven holdings,
sector-neutral construction, value-weighting, borrow costs, capacity analysis,
or Alphalens-style tear sheets.
