What This Project Shows

This project builds a leakage-controlled financial NLP pipeline over 500 SEC 10-K filings and tests whether disclosure text predicts future realized volatility. The preregistered Ridge model achieves a Rank IC of 0.2606, with claims bounded as prediction evidence rather than tradable alpha.
